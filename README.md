# 关于本mod

**Re:boot Storyteller**是一个叙事者mod，它本质上是一个同人创作，相关图像素材均已得到原型的本人同意。mod添加了五位新的叙事者，这五位叙事者在设计预期上面向不同时期的玩家，由于是我开发的第一个mod，因此在平衡性方面有所欠缺，如果mod存在什么问题或有什么好的点子，还请告诉我。

## 分发

目前，mod会在本平台（Github）发布beta（测试版）和正式版，Steam创意工坊只会发布正式版。同时，截至目前（2026年02月11日），由于一些原因，mod的C#源码暂时不予公开，具体公开时间……得取决于多久有人可以“给出络亚Loya感兴趣的数据”。

mod区分0Harmony版本（也就是带0Harmony后缀的版本）和普通版本，二者唯一的区别是前者不需要额外安装Harmony Mod作为前置。

## 叙事者们的主要设计以及完成度

接下来是对五位叙事者的简单介绍，这一小节只会简短描述五位叙事者的功能和设计预期。

### 「温和」斯瑞Sthri

基本属性：

+ 预期难度：低
+ 特性：心情奖励·大师（传奇）物品
+ 机制：？？

「Re:boot No.003」

来自亚美利戈的一个出逃家政机器人，在逃亡的途中被拉入到REBOOT_SYSTEM。斯瑞是一个和平主义者，通常不会给予太难的挑战，她总会留出充足的时间让你开拓建设。不过……如果打算用一些能让她崩溃的手段干掉她，那你就完蛋了。值得一提的是，斯瑞很喜欢画画，也许你可以试试搓出大师级的东西？

### 「稚刃」未绫V0Rei

基本属性：

+ 预期难度：中等（偏低）
+ 特性：战斗增援·肉体增幅
+ 机制：暗杀

「Re:boot No.000」

来自炽国的一位肯特奈什，目前是穷奇组织的专业杀手。由于过于强烈的想要家人的愿望，她被REBOOT_SYSTEM选中。她给予的挑战通常都会维持在正常稍微偏下的水平。她安排的事件通常比较贴合殖民地发展水平，并且在有限的幅度范围内稍稍为你放一点水。未绫不喜欢吵闹，她安排袭击时会着重考虑殖民地人口，在此基础上，她可能还会干一些本职工作。最后：她偶尔会给点称不上是帮助的战斗援助，要是吃到了副作用……后果自负。

### 「戏剧」海露美Helmi

基本属性：

+ 预期难度：中等（随机）
+ 特性：艺术加护
+ 机制：？？

「Re:boot No.006」

谁也不知道她的身世，可能只有收取她记忆，令其上岸的REBOOT略知一二？她对歌声与艺术非常在意，她偶尔会给你的殖民者一些艺术上的帮助。海露美喜欢变数和戏剧，在有限的范围内，海露美会为挑战或增幅或放水，增减幅度完全看她心情。在这个由她主导的边缘世界，让她发怒的方法还真是没有。但也许……用了你不该用的力量，会招致她头上海神大人的疯狂报复？

### 「制衡」文图拉Ventura

基本属性：

+ 预期难度：中等（偏高）
+ 特性：均衡象位
+ 机制：概率性心情崩溃干预

「Re:boot No.002」

她是REBOOT四人中唯一拥有全程“被绑架”记录的Player，大概是单纯倒霉被REBOOT看上了？她一直致力于逃脱REBOOT，文图拉并非武力派，通常以智识解决问题的她，对于你殖民者的遭遇，倒是能给一些科技上的援助。但代价是什么呢？文图拉心若止水，但擅长占卜的她，非常看重周期律和均衡问题，有关后者：她会将占卜介入到叙事中。七种象位将会如何影响叙事呢？那就得靠你自己去探索了。还有一件事，心情崩溃确实糟糕，但在文图拉的叙事下，这种问题恐怕不会是什么问题。
